# Nairobi Climate Time Series Analysis

Exploring 170 Years of Temperature Data (1850–2020)

## Overview
This repository contains a time series analysis of historical climate data 
for Nairobi, Kenya, examining long-term temperature trends and seasonal 
patterns across 170 years of recorded data.

## What This Project Explores
- Long-term warming trends in Nairobi's climate
- Seasonal temperature patterns and anomalies
- Decade-by-decade shifts in average temperatures
- Visual storytelling of climate change at a local African city level

## Key Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## Outputs
- Time series visualisations of temperature trends
- Seasonal decomposition plots
- Trend analysis highlighting Nairobi's warming trajectory

A narrative version of this analysis is published as a 
[Medium article](https://medium.com/@mariam.haji).

## Data Source
Historical climate data sourced from publicly available meteorological records.
